我们要学Python嘛，先来把它下好。首先我们要进入到Python的官网，
很简单，不知道网址，那就百度一搜，除开前面给了钱的广告，第一个
或者前几个就有我们的Python官网，然后找到我们的下载界面。这里呢
Python有多个版本可选，出于个人习惯，我这儿选用的是Python3.7.2
这个版本，大家可以跟我选择同一版本的。然后安装就基本是傻瓜式的
了，主要这儿我再解释下一些安装过程中的选项。我们后面的可选项基
本都是全选，这儿"Documentation" 表示安装Python的帮助文档。
"pip" 表示安装Python的第三方包管理工具，这个很重要的，后续我们
需要用到的第三方包几乎所有都是用pip工具来进行一句命令安装的。
包括卸载也是一样，都是一句命令完成。这儿也是勾选全部选项，然后
选择我们的一个安装路径。

下一个很重要的是配置环境变量，环境变量其中的环境，是指整台电脑
的一个运行环境，而变量是计算机中一段数据或值的抽象。通俗来理解
变量就是用来表示一段数据或值的。而我们下载的Python，其中里面的
Python.exe就是我们用来翻译Python程序的解释器，它能把Python代码
翻译成计算机能看懂的指令以供计算机执行。而我们怎么去引用我们的
Python.exe呢？在它的路径下，我们当然可以去引用到，只需要在我们
命令行中输入Python就行。但是一旦不是它所在的路径，那么我们计算
机是不认识Python是表示的什么的，因此我们要把Python.exe所在路径
设置在计算机的全局变量中。方法很简单，鼠标右键桌面上的此电脑，
选择属性，这儿找到我们的高级系统设置，选择环境变量，然后在我们
的系统变量中双击Path来编辑，点这儿的新建，输入我们的Python.exe
所在路径。然后我们在我们计算机中任意路径下执行Python试试，看吧
是不是已经有了。输入exit()或者Ctrl+C来退出。

接下来来下载我们的Python编译器，这个编译器说直白点就是我们一个
写代码的软件，那么就是哪个方便、哪个好用、哪个顺手就用哪个。
其实严格意义上，这些个只能算是编辑器，就和我们的文本文档一样。
执行代码翻译工作的还是我们的Python解释器，即我们的Python.exe
这儿我就推荐一个轻便好用的VS Code编译器，至于它相比于其他编译器
有什么值得我特别推荐的地方，我们后续用到再说，反正相信我，这真是
一款特别好用的编译器。我们还是找到它的官网，选择这个stable稳定版
的来进行下载。
